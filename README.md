# CryptoTracker

A Python-based Cryptocurrency Price Tracker developed using Selenium, WebDriver Manager, and real-time data scraping.

## Features
- Live price extraction of cryptocurrencies
- Automated browser using Selenium
- Error handling with retries
- Saves output or displays directly

## Technologies
- Python
- Selenium
- WebDriver Manager

## How to Run
1. pip install selenium
2. pip install webdriver-manager
3. python crypto_tracker.py
